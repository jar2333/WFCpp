Detailed Description

Tile Data Class.

Author
Ricardo Figueroa

Contains all the required TileData retrieved through the extraction. Most of the names are self explanitory. Id, width, height, pixels... The different sides (north, south, east, west) however refer to the list of pixels that encompase those sides for the tile. The side constraints however, contain a set of ids mapped to the id of the tile.
