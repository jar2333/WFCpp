\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classBMPImage}{BMPImage}} class. 

This class supports the import, export and basic modification of images as BMP files. It provides an interface for image processing in \mbox{\hyperlink{classExtractor}{Extractor}} and \mbox{\hyperlink{classSynthesizer}{Synthesizer}}.

At the time, we only support png and bmp format images.
