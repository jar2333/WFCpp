\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classSynthesizer}{Synthesizer}} class. 

This class implements the functionality of synthesizing grid into an out put image. At the moment, only png and bmp images are supported. All generated images are stored as \mbox{\hyperlink{classBMPImage}{BMPImage}} and transformed into specified format.
