\doxysubsection{Detailed Description}
\mbox{\hyperlink{classTile}{Tile}} Data Class. 

\begin{DoxyAuthor}{Author}
Ricardo Figueroa
\end{DoxyAuthor}
Contains all the required \mbox{\hyperlink{classTileData}{Tile\+Data}} retrieved through the extraction. Most of the names are self explanitory. Id, width, height, pixels... The different sides (north, south, east, west) however refer to the list of pixels that encompase those sides for the tile. The side constraints however, contain a set of ids mapped to the id of the tile. ~\newline
